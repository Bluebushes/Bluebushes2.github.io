A Calculator

For all your needs

But why a Calculator?

Since a calculator is a basic standard for many coding programs. It helps the programmer get a good grasp on how to work the math of a new programming language. In this case it was Javascript. Creating this calculator also told me how to overcome some hard obstacles, such as being able to use the same buttons to fill two different number values. Also I needed to have a 'fake' number value to display itself on the calculator screen, and then I used two real number values that were behind the screen math processed by Javascript. Lastly, I had to figure out how to let the user continously calculate after they had an answer. Either way, I finally made this calculator, so enjoy the use of this useless calculator.




Purpose of this Calculator

The same purpose of every single other calculator that you probably have and have utilized: to calculate numbers. So what else did you think the calculator was for? Thought there was some easter egg? Well no, there isn't one. I would be too lazy to make on anyways. So enjoy using this calculator to calculate numbers. Design of the Calculator

I used some Flexbox with CSS, but it does turn out that the space around each button is quite wide. Not sure how to fix that, but the calculator will do for now. I do wish flexbox could give you an option to change spacing of 'space-around' and 'space between', but hey, it's still very powerful and does basically what you imagine it would do. I also added box-shadows to each button to make it look 3D when pressed. Oh, and the button colors changed too, they look much more snazzy than just gray *cough cough* "grey".




Uses of this "Calculator"

Maybe use it to count, maybe try to make words out of the number, maybe find the easter egg, maybe to just calculate with this terrible calculator if you somehow feel like calculating with it. But either way, I don't know what you can do with the calculator, as long as you can press the buttons and decide what to do with the numbers on the screen.
